About

Object classifier web app:The Flow of

With our web app a user can input his/her image or just turn on the webcam to see live object classfication in the browser with just a click! A bounding box with the label and classification score will be produced on the video feed itself, making it visually obvious the kind of objects an image or live webcam feed has. not only that our web cam app can also be used by visually impaired users, as it also uses the speech synthesis API which is inbuilt in almost every modern browser which also helps the user.

Animation

Animation for our project is done by using an open source library called as p5.js (A product from the processing foundation), It is a very powerful library which makes it easy to do animation on the web browser with the power of javaScript, all the animations in this project were done using p5 only

Machine Learning

The machine learning was the heart of this project, it was possible with the help of another open source library called as the ml5.js, which is an abstraction over a very famous machine library called as the tensorflow.js. This library abstracts over a lot of stuff of tensorflow.js, making it easier to implement machine learning on browser easy and fast.

Front end

The front end for the websites was done using two libraries / framework. the first is bootstrap and the other is jQuery. bootstrap makes it easier to make webpages by adding predefined classes to your HTML elements, and jQuery makes it easier to make websites dynamic.
